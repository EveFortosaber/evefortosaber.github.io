# Note Of Markdown

# 1. 准备（Preparation）

  • 应用软件：Video Studio Code
  • 工作文件：makedown (后缀为 .md ）

# 2. 开始（Beginning）

  • 创建一个文本文档更改后缀为 .md 然后用 Visual Studio Code 打开，或者先打开 Visual Studio Code 然后新建一个后缀为 .md 的文件。
  • 打开侧边预览
在view中打开 Command Palette 然后输入Open Preview to the Side

# 3. 语法（Grammar）

# 标题（Title)
  1. 标题可分为一到六级分别用 #、##、###...... 以此类推
    其中 #表示的最大，###### 最小。语法如下：
# 标题1
## 标题2
### 标题3
...
  1. 换行直接回车
  2. 字体
    • *内容* 为斜体
    • **内容** 为加粗
    • ***内容*** 为粗斜体
    • ~~内容~~ 为删除线
      表现：内容、内容、内容、内容
